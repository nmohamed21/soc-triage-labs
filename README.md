# SOC Triage Labs

This repository contains hands-on Security Operations Center (SOC) triage labs demonstrating alert analysis, log correlation, and incident response workflows commonly used in blue team environments.

## Lab 01 – Authentication Log Triage
Python-based analysis of authentication logs to detect potential brute-force login activity by correlating failed login events across multiple users and source IP addresses.

## Lab 02 – Network Traffic Triage
- Analyzed SMB and RPC network traffic using packet-level inspection
- Identified indicators of potential lateral movement activity
- Detected SAMR-based account enumeration attempts
- Documented findings using SOC incident response and triage methodologies

## Tools Used
- Python
- Wireshark
- Windows authentication-style logs

## Skills Demonstrated
- SOC alert triage and prioritization
- Log analysis and event correlation
- Network traffic analysis
- Threat hunting fundamentals
- Incident investigation and reporting
- SOC investigation workflows
